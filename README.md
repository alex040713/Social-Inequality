# Social Inequality

## Project Overview
There are many theories about how social inequality affects the general life satisfaction of the population, health, and crime.
• Experts also warn that social inequality leads to an erosion of democratic values.
• This project aims to examine how social inequality has developed over time in different countries and between different countries. It also aims to examine how social inequality is linked to democratic values, life satisfaction among the population, and crime.

## Repository Structure
The project is organized into the following directories to ensure reproducibility:

* **`/Data`**: Contains the raw datasets and any processed data files used in the analysis.
* **`/Program`**: Includes all source code, scripts (e.g., Python `.ipynb` or R scripts), and data cleaning procedures.
* **`/Results`**: Stores output figures, statistical tables, and model summaries generated by the code.
* **`executive_summary.pdf`**: A high-level summary of the research findings and policy implications.
* **`Social Inequality new.pptx`**: The final presentation deck for this project.
* **`Social Inequality old.pptx`**: The initial presentation deck for this project.

## Getting Started
1. **Prerequisites**: Ensure you have R installed.
2. **Installation**: Clone this repository and install required libraries:
   `pip install -r requirements.txt`
3. **Execution**: Run the scripts located in the `/Program` folder to reproduce the analysis.
